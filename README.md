# HaulOps Fleet Management System

Professional fleet management dashboard for a 20-truck trucking company.

## Run

Open `public/index.html` directly in a browser, or run the optional built-in server:

```bash
node server.js
```

Then visit `http://localhost:4173`.

## Included

- Responsive dashboard with cards, charts, analytics, alerts, search, and filters
- Truck CRUD with realistic data for 20 trucks
- Diesel, maintenance, tire, driver, expense, report, alert, and analytics modules
- Light and dark modes
- CSV export for Excel, JSON export, and print/PDF-ready reports
- Local persistence through browser localStorage
- Dependency-free Node server for deployment or local preview
